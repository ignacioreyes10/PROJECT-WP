# ErasmusHustlers

## Description
ErasmusHustlers is a web application that allows users to search and book spaces such as apartments, houses, and venues.

## Features
- User registration and login
- Property search and filtering
- Property details
- Booking system
- Admin panel

## User Stories
- As a user, I want to register and log in so that I can access the platform.
- As a user, I want to search and filter properties so that I can find the best option.
- As a user, I want to book a property so that I can reserve it easily.
